Merci

-------------------

Load and update plugins submodule::

    git submodule init
    git submodule update --recursive
    git submodule status

Install virtualenv::

    (sudo) pip install virtualenv

To build the site::

    make html

To serve it on http://localhost:8000::

    make serve

and to publish it::

    make publish

Publish to afpy server::

    make afpy
